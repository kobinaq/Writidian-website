# Writidian

Marketing site for [Writidian](https://app.writidian.com) — a writing sanctuary for intentional writers.

## Stack

- Next.js (App Router), TypeScript, Tailwind CSS
- GSAP ScrollTrigger (hero scrub + section reveals)
- Motion (CTA micro-interactions)
- Web Audio meditation pad in the hero (placeholder until a licensed loop is supplied)

## Develop

```bash
npm install
npm run dev
```

Primary CTA links to `https://app.writidian.com`.

## Client open items

- Free vs paid tiers and when paid kicks in
- Whether prompts/soundscapes are genre-specific (site copy currently reflects genre soundscapes + daily prompts)
- Whether audio is licensed, original, or generated (hero uses a synthesized pad for now)
- Whether there is a social/community layer
